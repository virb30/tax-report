# CH-03: Verify a casual user can generate a year-end report, understand pending data, and copy declaration text without confusion

**Mode:** Charter-With-Tour
**Persona:** Casual User
**Surface:** Relatorio Bens e Direitos
**Entry URL:** http://localhost:5173
**Tour:** Feature
**Time-box:** 60 minutes

## Mission

Find any point where the report screen claims readiness or copyability while the user still lacks the information needed to declare assets confidently.

## Out of Scope

- Tax math deep audit
- Real Receita Federal submission
- Browser-specific clipboard quirks beyond observable UI feedback

## Must Try

- Generate the report for the default base year
- Distinguish ready, pending, and unsupported items
- Trigger a copy action and observe success or fallback feedback

## Must Avoid

- Using production taxpayer data
